# Claude Skills

Behavior-change Skills for Claude.ai — free to download, no sign-up required.

---

## What are Claude Skills?

Claude Skills let you save a set of instructions that Claude automatically applies in conversations. Instead of repeating the same prompt every time, a Skill activates when it recognizes the right context — and intervenes without you having to ask.

Most people use Skills to standardize output: presentations, plans, reports. These Skills take a different approach: **behavior change**.

The idea is simple. Self-help books fail not because the content is wrong, but because awareness shows up at the wrong moment. You read it in the evening when you're calm. The pattern plays out during the day under pressure. A proactive Skill bridges that gap — it recognizes the pattern and intervenes when it matters.

---

## Available Skills

### behavior-skill-builder
Helps you build a personalized behavior-change Skill based on any book, model, or methodology.

The Skill conducts a thorough interview — your motivation, recurring situations, avoidance patterns, rationalization phrases, desired behavior — and generates a ready-to-upload `.skill` file tailored to your specific situation.

**[Download behavior-skill-builder.skill](https://github.com/puckdehondt-bonana/claude-skills/raw/main/behavior-skill-builder.skill)**

---

## How to install a Skill

1. Download the `.skill` file
2. Open [Claude.ai](https://claude.ai)
3. Go to **Settings → Skills**
4. Click **Upload Skill** and select the file
5. The Skill is now active in your conversations
